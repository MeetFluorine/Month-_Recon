How quantities are computed
  • Inward: sums NonSerialised_Inward_Qnty for bulk items, and counts one unit per row for serial-controlled items (each serial number = 1 inward unit).
  • Issuance: sums Issued_Qty. Use the Allocate / Recover toggle in the dashboard to include or net-off returned material.
  • S2S Transfer: reads the two sheets in the workbook (one per transfer direction), sums the transfer quantity per row, and works out the month from the date column on each row rather than the sheet name. Use the "Flow" toggle in the dashboard to switch between directions — a workbook's sheets are matched by name/columns automatically, and if a sheet's headers don't look like item code / store / quantity / date, it's skipped and you'll see which one so we can fix the column matching together.
  • Only the columns needed for aggregation are read from each row, and each sheet is discarded from memory right after it's summarised — so even multi-lakh-row workbooks stay light once parsing gets going.
  • For a 70–110MB workbook, expect roughly 30 seconds to 2 minutes to open the file before the per-sheet progress starts moving — that first stretch is the spreadsheet engine unpacking the workbook, and it's normal for it to look quiet.
  • Everything happens locally in this browser tab; the file is never uploaded anywhere.
  • Detail table: the "Detail — month × item × store" table (bottom of the dashboard) now also shows MO/IR Number, Ordered Qnty, NonSerialised Inward Qnty and Serialised Inward (True/False) for every mode — Inward, Issuance and S2S. These are matched from your workbook's own column names automatically (e.g. "MO/IR Number", "Ordered Qty", "Serial No"); if a workbook genuinely doesn't have one of these columns, that cell just shows "—".
  • Calculation dashboard: upload any or all three reports above — each stays loaded even after you switch mode and upload another — then hit "Open Calculation Dashboard" to reconcile a single item code at a single store: (MRS Inward + S2S in from other stores) minus (Issuance to FE + S2S out to other stores) = remaining stock, with a downloadable .xlsx of the full breakdown.
